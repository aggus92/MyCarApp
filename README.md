# MyCarApp

**Aplikacja do zarządzania pojazdami**

**Cel aplikacji**

Celem projektu jest stworzenie aplikacji umożliwiającej zarządzanie pojazdami. 

**Funkcjonalność**

- rejestracja użytkowników,
- obsługa wielu pojazdów,
- zarządzanie wydatkami,
- generowanie raportów,
- statystyki wydatków,
- przypomnienia.
